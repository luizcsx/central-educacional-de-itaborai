# Guia de Contribuição

Agradecemos o interesse em contribuir com o projeto do Centro Educacional de Itaboraí. Para manter a organização e a qualidade do código, solicitamos que siga estas diretrizes.

## Como contribuir

1. Realize o Fork do projeto.
2. Crie uma Branch para sua modificação (git checkout -b feature/NovaFuncionalidade).
3. Mantenha o padrão de nomenclatura e a estrutura de pastas existente.
4. Certifique-se de que o código está devidamente comentado em português.
5. Envie um Pull Request detalhando as alterações realizadas.

## Padrões de Código

* Use identação de 2 espaços.
* Nomes de variáveis e funções devem ser claros e em português ou inglês, mantendo a consistência do arquivo original.
* Evite subir arquivos de configuração local ou chaves de API (.env).

## Comportamento

Este é um ambiente educacional. Mantenha o respeito nas discussões de issues e revisões de código.
